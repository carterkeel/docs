## Headings
```md
# H1
## H2
### H3
#### H4
##### H5
###### H6
```

## Emphasis
```md
*italic* or _italic_  
**bold** or __bold__  
***bold italic***  
~~strikethrough~~
```

## Lists

### Unordered List
```md
- Item 1
- Item 2
  - Subitem 2.1
* Item 3
```

### Ordered List
```md
1. First
2. Second
   1. Subitem
```

## Links
```md
[Text](https://example.com)
[OpenAI](https://openai.com)
```

## Images
```md
![Alt text](https://via.placeholder.com/150)
```

## Code

### Inline Code
```md
`inline code`
```

### Code Block
\`\`\`language
function greet() {
  console.log("Hello, world!");
}
\`\`\`

## Blockquotes
```md
> This is a quote.
>> Nested quote.
```

## Horizontal Rule
```md
---
***
___
```

## Tables
```md
| Syntax | Description |
|--------|-------------|
| Header | Title       |
| Cell   | Text        |
```

## Task Lists
```md
- [x] Task complete
- [ ] Task incomplete
```

## Escaping Characters
Use a backslash `\` to escape Markdown characters:
```md
\*escaped asterisk\*
```
